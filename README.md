# Webpack TypeScript

[![Build Status](https://travis-ci.org/silbinarywolf/webpack-typescript.svg?branch=master)](https://travis-ci.org/silbinarywolf/webpack-typescript)

## Requirements

* Node v10.16.0+
* Yarn 1.17.3+
* Git 2.19.2+
** This is required for proper support of symlinks on Windows
